# Banking Application (Windows Forms)

A simple, interactive desktop-based Banking Application built using C# and Windows Forms. This application serves as a digital ledger to manage customer profiles, generate unique account details, and process everyday financial transactions.

## Features
* **Automated Account Creation:** The system automatically generates a unique Customer ID and Account Number upon registration.
* **Transaction Management:** Built-in operations to seamlessly handle money **Deposits** and **Withdrawals**.
* **Data Retrieval:** A dedicated function to search, fetch, and display existing customer information instantly.
* **Input Resetting:** A single-click clear feature to reset all text fields for the next transaction.

## Built With
* **Language:** C#
* **Framework:** .NET / Windows Forms (WinForms)
* **IDE:** Visual Studio / VS Code

## File Structure Overview
* `Form1.cs`: Contains the core backend logic, button click event handlers, and transaction algorithms.
* `Form1.Designer.cs`: Houses the auto-generated layout code for the user interface elements.
* `Program.cs`: The main entry point that launches the application window.

## Getting Started
1. Clone this repository to your local machine.
2. Open the `BankingWinAppForms.sln` solution file in **Visual Studio**.
3. Press **F5** or click the **Start** button to compile and run the application.
